# WhisperTranscriber

**WhisperTranscriber** is a **Graphical User Interface (GUI)** tool that uses **OpenAI's Whisper model** to convert audio files (MP3 and WAV) into text and save the output directly as a **Word document (.docx)**.

---

## Features

- Supports **MP3 and WAV audio files**  
- Uses **OpenAI Whisper** for highly accurate speech-to-text conversion  
- **Two Progress Bars**:  
  - Model download progress  
  - Audio transcription progress  
- **Cancel button** to stop ongoing operations  
- **Modern and user-friendly GUI** built with CustomTkinter  
- Save output directly in **Word (.docx)** format  

---

## Supported Languages

Whisper supports multiple languages for speech-to-text transcription. Currently, **WhisperTranscriber supports all languages that the Whisper model can recognize**, including but not limited to:

- English  
- Spanish  
- French  
- German  
- Italian  
- Portuguese  
- Dutch  
- Russian  
- Chinese  
- Japanese  
- Korean  
- Arabic  

> Note: Accuracy may vary depending on the language and audio quality.
